# ♟️ Xadrez Console — Jogo de Xadrez em C#

![C#](https://img.shields.io/badge/C%23-7F5AF0?style=flat&logo=csharp&logoColor=white)
![.NET](https://img.shields.io/badge/.NET-Console_App-512BD4?style=flat&logo=dotnet&logoColor=white)
![Status](https://img.shields.io/badge/Status-Concluído-22c55e?style=flat)

Jogo de xadrez completo rodando no terminal (**CMD**), desenvolvido em **C#** com foco em lógica de programação orientada a objetos, herança, polimorfismo e estruturas de dados.

---

## 🎮 Funcionalidades

- ♟️ Jogo completo de xadrez para 2 jogadores no terminal
- 🎯 Movimentação de todas as peças com regras oficiais
- ⚠️ Detecção de **Xeque** e **Xeque-Mate**
- 🔄 Controle de turno entre jogadores
- 🗺️ Exibição do tabuleiro atualizado a cada jogada
- 🚫 Validação de movimentos inválidos

---

## 🛠️ Tecnologias

- C# (.NET)
- Aplicação Console
- Orientação a Objetos (POO)

---

## 📐 Conceitos Aplicados

- **Herança** — classe base `Peca` herdada por cada tipo de peça
- **Polimorfismo** — cada peça implementa seus próprios movimentos
- **Encapsulamento** — lógica de validação isolada por responsabilidade
- **Matrizes** — representação do tabuleiro 8x8
- **Exceptions** — tratamento de jogadas inválidas

---

## 🚀 Como Executar

### Pré-requisitos
- [.NET SDK](https://dotnet.microsoft.com/download)

### Passos

```bash
# Clone o repositório
git clone https://github.com/VanessaCarvalho-VNS/Xadrez-Console.git

# Acesse a pasta do projeto
cd Xadrez-Console

# Execute o projeto
dotnet run
```
